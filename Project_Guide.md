# الدليل الشامل لمشروع FP-Growth 🛒

أهلاً بيك يا بطل في مشروعك! الدليل ده مكتوب مخصوص عشانك بلهجة بسيطة جداً، كأننا قاعدين مع بعض وبشرحلك خطوة بخطوة إزاي تشغل المشروع وتفهمه كويس جداً عشان المناقشة.

---

## 1. إزاي تجهز جهازك (Environment Setup) 🛠️
عشان نشغل المشروع، محتاجين حاجتين أساسيتين:

1. **Python (بايثون):**
   - ادخل على الموقع الرسمي: [python.org/downloads](https://www.python.org/downloads/)
   - حمل أحدث إصدار للويندوز.
   - **خطوة مهمة جداً وأنت بتسطب:** هتلاقي مربع تحت مكتوب عليه **"Add Python to PATH"**.. لازم لازم تعلم عليه صح (✔️) قبل ما تدوس Install Now.

2. **VS Code (فيجوال ستوديو كود):**
   - ده البرنامج اللي هنشوف منه الكود.
   - حمله من هنا: [code.visualstudio.com](https://code.visualstudio.com/) وسسطبه عادي جداً (Next -> Next).

---

## 2. تسطيب المكتبات (Libraries) 📚
عشان البرنامج يشتغل، محتاجين ننزل شوية "مكاتب" (أكواد جاهزة بتساعدنا). 
- افتح الـ VS Code.
- من القائمة اللي فوق اختار `Terminal` وبعدين `New Terminal`.
- اكتب الأمر ده وادوس Enter (لازم تكون متصل بالنت):
```bash
pip install streamlit pandas mlxtend
```

**إيه لزمة المكاتب دي؟**
- `pandas`: دي بنستخدمها عشان نقرأ ملفات الإكسيل والـ CSV ونرتب البيانات في جداول.
- `mlxtend`: دي المكتبة اللي فيها كود الـ FP-Growth جاهز ومكتوب صح.
- `streamlit`: دي الأداة السحرية اللي عملنالك بيها واجهة المستخدم (الزراير والألوان) عشان البرنامج ميبقاش مجرد شاشة سودا.

---

## 3. إزاي تشغل المشروع (Running the Project) 🚀
في الـ Terminal بتاع VS Code، اتأكد إنك جوه الفولدر بتاع المشروع (`fp-projj`) واكتب الأمر ده:
```bash
streamlit run app.py
```
أول ما تدوس Enter، هتلاقي متصفح النت (جوجل كروم مثلاً) فتح لوحده وظهرتلك واجهة البرنامج الجاهزة!

---

## 4. البيانات اللي هنجرب بيها (Dataset) 📊
في المشروع، إنت عندك 3 طرق عشان تدخل الداتا:
1. **Upload CSV:** ترفع ملف داتا جاهز.
2. **Use Sample Dataset:** دي عينة أنا مجهزهالك (`sample_dataset.csv`) عشان تجرب بيها علطول.
3. **Manual Input (إدخال يدوي):** ده أوبشن جديد ضفناه بيسمحلك تكتب الـ Transactions بنفسك بإيدك سطر سطر (تكتب المنتجات وتفصل بينها بفاصلة أو مسافة).

**لكن لو حبيت تجيب داتا سيت تانية للرفع، تجيبها منين؟**
- **موقع Kaggle:** ده كنز الداتا. ادخل على [Kaggle.com](https://www.kaggle.com/) واعمل بحث عن `Market Basket Analysis dataset` أو `Groceries dataset`. هتلاقي ملفات كتير نزلها.
- **إزاي تعمل ملف إكسيل بنفسك للرفع؟**
  1. افتح الإكسيل.
  2. اعمل عمودين: الأول سميه `Transaction_ID` والتاني سميه `Items`.
  3. في الـ Items، اكتب الحاجات اللي اشتراها العميل مفصولة بمسافة أو فاصلة (مثلاً: `Milk Bread Eggs`).
  4. اعمل Save As واختار الصيغة `CSV (Comma delimited)`.
  5. ارفع الملف ده في البرنامج بتاعنا من اختيار `Upload CSV`.

## 5. الشرح المبسط للخوارزمية (Explanation) 🧠
الـ FP-Growth ده عبارة عن إيه؟
تخيل إنك صاحب سوبر ماركت، وعايز تعرف **"الناس اللي بتشتري كذا، غالباً بيشتروا معاه كذا"**. يعني مثلاً اللي بيشتري "حفاضات"، غالباً بيشتري معاها "لبن". 

عشان نعمل ده بنحتاج شوية مصطلحات:
- **Frequent Itemset (المجموعات المتكررة):** دي المنتجات اللي بتتباع مع بعض كتير. زي (العيش واللبن).
- **Support (الدعم أو نسبة الظهور):** ده رقم بيعبر عن "المنتجات دي ظهرت في كام فاتورة من إجمالي الفواتير؟". يعني لو عندنا 10 فواتير، و(العيش واللبن) موجودين في 4 فواتير، يبقى الـ Support بتاعهم 40% (أو 0.4). في البرنامج إنت بتحدد الـ Min Support، يعني بتقوله "هاتلي بس المنتجات اللي ظهرت بنسبة كذا أو أكتر".
- **FP-Growth:** دي الطريقة (الخوارزمية) اللي بتدور بيها على المجموعات المتكررة دي. بدل ما تقعد تلف على كل الفواتير ميت مرة، هي بتعمل "شجرة" (Tree) في الذاكرة بترتب فيها المنتجات، وبكده بتلاقي النتائج بسرعة جداً جداً مقارنة بالطرق القديمة زي (Apriori).

## 7. أخطاء شائعة وإزاي تحلها (Common Errors) ❌
1. **"python is not recognized" أو "pip is not recognized":** 
   - الحل: إنت نسيت تعلم على "Add Python to PATH" وإنت بتسطب بايثون. امسحه وسطبه تاني وعلم عليها.
2. **"No module named pandas/streamlit":**
   - الحل: إنت مسطبتش المكتبات. ارجع للخطوة رقم 2 واكتب أمر الـ `pip install`.
3. **"No frequent itemsets found":**
   - الحل: دي مش مشكلة في الكود، ده معناه إن الـ Support اللي إنت محدده عالي جداً. قلل المؤشر بتاع الـ "Minimum Support" شوية (مثلاً خليه 0.1) ودوس Run تاني.
